# AirQ

## How to run locally

Clone this repository.
Navigate into the project folder

Run the following commands:

### Using npm
```bash
npm install 
npm run dev
```

### Using yarn
```bash
yarn install 
yarn run dev
```

## Notes

* [NextJS]('https://nextjs.org/')
* [Chakra-UI]('https://chakra-ui.com/')
